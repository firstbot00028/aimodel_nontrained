# 🧠 Adam-AI: Build Your Own LLM from Scratch
### Managed by: Engineer Adam 💎

This repository contains the complete source code to build and train a Small Language Model (SLM). Instead of giving a pre-trained model, I am opening the "Brain Logic" so the community can train it on their own custom data.

## 🚀 Why this Repo?
- **Learn AI Internals:** Understand how Transformers, Embeddings, and Self-Attention work.
- **Custom Training:** Train the model on your own text (Malayalam, Python code, etc.)
- **Lightweight:** Designed to run on consumer-grade hardware.
